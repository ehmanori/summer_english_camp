# 가방별 준비물 배치표

Version: v1.1

작성일: 2026-07-21

대상: 아버지 + 윤찬 + 소윤 (3인)

이 문서는 `28인치 캐리어 2개`, `20인치 캐리어 1개`, `배낭 1개`에 준비물을 실제로 나눠 넣기 위한 배치표다. 캐리어 하나의 `몸통`과 `뚜껑`을 별도 구획으로 보아 총 6개 구획을 사용한다.

숙소 도착 후 짐을 모두 꺼내 정리하므로 캐리어 안의 완벽한 분류보다 `위탁 가능 여부`와 `두 28인치의 무게 균형`을 우선한다.

## 1. 가방 역할과 운송 구분

| 가방 | 역할 | 운송 |
|---|---|---|
| 28인치 A | 가족 의류와 무게 분산용 신발·공용품 | 위탁수하물 |
| 28인치 B | 최소 생활·세면·캠프용품 | 위탁수하물 |
| 20인치 C | 전자기기와 기내·도착 직후 사용할 물품 | 기내 반입 |
| 배낭 D | 여권·현금·휴대폰·배터리·필수 약 | 보호자가 항상 휴대 |

핵심 원칙:

- 아래 구획 배치는 시작안이며, 최종적으로는 무게를 재면서 A와 B 사이에서 옮긴다.
- `28인치 A`에 가족 옷과 신발·우산을 함께 넣어 무게를 분산한다.
- `28인치 B`에는 한국에서 꼭 가져갈 최소 생활·캠프용품만 넣는다.
- 두 28인치의 실제 무게 차이는 `2kg 이내`로 맞춘다.
- 한국 멀티탭 1개와 Type G 변환 플러그 2개를 가져간다.
- Type G 1개는 멀티탭을 연결한 공용 충전 거점에, 나머지 1개는 침대 옆 단독 충전에 사용한다.
- 멀티탭은 휴대폰·태블릿·노트북 충전용으로만 사용하고 드라이기·전기포트 같은 고출력 제품은 연결하지 않는다.
- 파손되기 쉽거나 위탁할 수 없는 물품은 `20인치 C`와 `배낭 D`에 넣는다.
- 20인치의 기내 반입 허용 크기·무게는 출발 전에 항공사 기준으로 최종 확인한다.

## 2. 여섯 구획 한눈에 보기

| 구획 | 위치 | 분류 |
|---|---|---|
| 1 | 28인치 A 몸통 | 아버지 의류·가족 신발·무거운 공용품 |
| 2 | 28인치 A 뚜껑 | 윤찬·소윤 의류·가벼운 공용품 |
| 3 | 28인치 B 몸통 | 최소 세면·건강용품 |
| 4 | 28인치 B 뚜껑 | 캠프·문구·가벼운 생활용품 |
| 5 | 20인치 C 몸통 | 태블릿·노트북과 충전 장비 |
| 6 | 20인치 C 뚜껑 | 기내 편의용품과 도착 직후 사용할 물품 |

## 3. 구획 1 — 28인치 A 몸통: 아버지 의류·무게 분산

- [ ] 반소매 상의 7~10벌
- [ ] 반바지·얇은 하의 5~7벌
- [ ] 속옷 7~10벌
- [ ] 양말 7~10켤레
- [ ] 잠옷 2벌
- [ ] 얇은 긴팔 또는 바람막이 1벌
- [ ] 수영복·수영모
- [ ] 가족의 여분 운동화·샌들
- [ ] 모자
- [ ] 아버지 중형 속건 수건 1장
- [ ] 공용 중형 속건 수건 1장
- [ ] 한국 멀티탭 1개

넣는 방법:

- 가족 신발은 바퀴 쪽에 둔다.
- 상의·하의·속옷을 각각 파우치로 나눈다.
- 도착 후 파우치째 옷장으로 옮길 수 있게 구성한다.

## 4. 구획 2 — 28인치 A 뚜껑: 아이들 의류

윤찬과 소윤의 물품을 색이 다른 파우치로 구분한다.

### 윤찬 파우치

- [ ] 상의·하의 7~10일분
- [ ] 속옷·양말 7~10일분
- [ ] 잠옷 2벌
- [ ] 바람막이 1벌
- [ ] 수영복·수영모
- [ ] 모자
- [ ] 윤찬 중형 속건 수건 1장

### 소윤 파우치

- [ ] 상의·하의 7~10일분
- [ ] 속옷·양말 7~10일분
- [ ] 잠옷 2벌
- [ ] 바람막이 1벌
- [ ] 수영복·수영모
- [ ] 모자
- [ ] 소윤 중형 속건 수건 1장

### 가벼운 공용품

- [ ] 접이식 우산 또는 우비
- [ ] 큰 세탁망 2개(상의·하의용)
- [ ] 작은 세탁망 2개(속옷·양말용)
- [ ] 세탁물 보관 가방

넣는 방법:

- `윤찬`, `소윤` 이름표를 파우치 바깥쪽에 붙인다.
- 속옷과 잠옷 파우치를 뚜껑 위쪽에 두어 첫날 쉽게 꺼낸다.

## 5. 구획 3 — 28인치 B 몸통: 최소 생활용품

액체류와 무게가 있는 물품을 바퀴 쪽에 둔다.

### 세면용품

- [ ] 칫솔·치약 3인분
- [ ] 샴푸·바디워시 등 첫 며칠분
- [ ] 선크림
- [ ] 벌레 기피제와 벌레 물린 뒤 바르는 약
- [ ] 개인 안경·렌즈용품
- [ ] 면도기 등 보호자 개인용품

### 건강용품

- [ ] 체온계
- [ ] 해열·진통제 예비분
- [ ] 소화제·지사제·감기약
- [ ] 밴드·소독용품
- [ ] 마스크 여분

주의:

- 액체류는 각각 지퍼백에 넣고 한 번 더 큰 방수 파우치에 넣는다.
- 비행 중 필요한 약은 위탁하지 말고 배낭에 넣는다.
- 샴푸·바디워시는 첫 2~3일분만 가져가고, 세제와 소모품은 RNF 도착 후 구매한다.
- 이 구획이 무거우면 선크림·세면용품 일부를 현지 구매 항목으로 돌린다.

## 6. 구획 4 — 28인치 B 뚜껑: 캠프·공용용품

- [ ] 등원 가방 2개
- [ ] 필통 2개
- [ ] 연필·지우개·필기구
- [ ] 노트
- [ ] 이름을 적은 텀블러 2개
- [ ] 학생증 목걸이 또는 케이스
- [ ] 이름표·네임펜
- [ ] 물티슈·휴지 여분
- [ ] 장바구니 또는 에코백
- [ ] 여분 지퍼백

넣는 방법:

- 등원 가방은 납작하게 펴서 가장 아래에 둔다.
- 윤찬과 소윤의 캠프 문구를 각각 지퍼백이나 필통으로 구분한다.
- 텀블러 안에 작은 문구나 양말을 넣어 공간을 활용한다.

## 7. 구획 5 — 20인치 C 몸통: 전자기기

20인치는 기내 반입을 전제로 한다.

- [ ] 학생용 태블릿 또는 노트북 2대
- [ ] 각 기기 충전기와 케이블
- [ ] 영국식 Type G 전용 변환 플러그 2개
- [ ] 이어폰 또는 헤드폰
- [ ] 기기용 보호 케이스

넣는 방법:

- 태블릿·노트북은 서로 맞닿지 않도록 완충재나 옷 한 벌로 분리한다.
- 케이블은 기기별 파우치로 나누고 이름을 붙인다.
- 보조배터리는 20인치가 아니라 배낭에 넣는다.

## 8. 구획 6 — 20인치 C 뚜껑: 기내·도착 직후 물품

- [ ] 아이별 얇은 겉옷 1벌
- [ ] 아이별 갈아입을 옷 1세트
- [ ] 물티슈·휴지·마스크
- [ ] 아이 간식
- [ ] 빈 물병 또는 텀블러 1개
- [ ] 간단한 놀거리
- [ ] 여권·보험·숙소·픽업 예약서 사본
- [ ] RNF 주소와 연락처 출력본

갈아입을 옷은 수하물 지연 대비가 아니라 비행 중 음식물 오염, 땀, 멀미 등에 대응하기 위한 것이다.

## 9. 배낭 D — 보호자가 항상 휴대

### 바로 꺼내는 칸

- [ ] 여권 3권
- [ ] 탑승권과 서류 파우치
- [ ] 휴대폰 3대
- [ ] 지갑과 주 사용 카드
- [ ] 볼펜
- [ ] 물티슈·휴지

### 보안 칸

- [ ] 예비 카드
- [ ] MYR·SGD 현금
- [ ] 레고랜드·반딧불 투어 인솔자 연락처와 비상연락 메모
- [ ] SGAC·MDAC 제출 확인본
- [ ] 여행자보험 증권
- [ ] JJTOUR 예약서와 RNF 주소

### 배터리·약 파우치

- [ ] 새 20,000mAh 보조배터리 1개(단자 보호 후 개별 지퍼백 보관)
- [ ] 휴대폰 충전 케이블 1세트
- [ ] 평소 복용약
- [ ] 아이용 해열제 등 이동 중 필요한 최소 상비약
- [ ] 멀미약·알레르기약
- [ ] 처방전 또는 영문 약 정보

## 10. 짐 싸는 순서

1. 바닥에 `1~6` 구획 표지를 놓는다.
2. 의류는 사람별 파우치로 나눠 구획 1과 2에 넣는다.
3. 세면·약·생활용품은 용도별 파우치로 나눠 구획 3에 넣는다.
4. 캠프 물품은 아이별로 구분해 구획 4에 넣는다.
5. 전자기기는 완충 포장해 구획 5에 넣는다.
6. 기내에서 쓸 물품은 구획 6과 배낭에 나눈다.
7. 위탁 금지·귀중품이 28인치 캐리어에 없는지 확인한다.
8. 캐리어별 무게를 재고 항공사 허용 기준 안인지 확인한다.
9. 28인치 B가 A보다 2kg 이상 무거우면 신발·우산·멀티탭·옷 일부를 A와 B 사이에서 옮기거나 현지 구매품을 뺀다.
10. 여섯 구획을 사진으로 찍어 휴대폰에 저장한다.

## 11. 위탁·기내 최종 확인

### 28인치 위탁수하물에 넣지 않을 것

- [ ] 여권·현금·카드
- [ ] 휴대폰·태블릿·노트북
- [ ] 보조배터리
- [ ] 필수 복용약
- [ ] 원본 예약·보험 서류

### 출발 전 확인

- [ ] 28인치 A에는 의류와 무게 분산용 신발·공용품이 정리되어 있음
- [ ] 28인치 B에는 최소 생활·캠프용품만 들어 있음
- [ ] 두 28인치의 무게 차이가 2kg 이내임
- [ ] 20인치는 기내 반입 허용 크기·무게 안에 있음
- [ ] 배낭에 여권·현금·카드·보조배터리·필수 약이 있음
- [ ] 액체류가 이중 포장되어 있음
- [ ] 모든 가방에 이름과 연락처가 표시되어 있음
