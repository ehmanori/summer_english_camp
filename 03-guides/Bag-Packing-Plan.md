# 가방별 준비물 배치표

Version: v1.0  
작성일: 2026-07-21  
대상: 아버지 + 윤찬 + 소윤 (3인)

이 문서는 `28인치 캐리어 2개`, `20인치 캐리어 1개`, `배낭 1개`에 준비물을 실제로 나눠 넣기 위한 배치표다. 캐리어 하나의 `몸통`과 `뚜껑`을 별도 구획으로 보아 총 6개 구획을 사용한다.

## 1. 가방 역할

| 가방 | 역할 | 운송 기준 |
|---|---|---|
| 28인치 A | 도착 직후 먼저 사용할 생활 짐 | 위탁수하물 |
| 28인치 B | 나머지 생활 짐과 캠프·공용 물품 | 위탁수하물 |
| 20인치 C | 위탁수하물 지연 시 24시간 버티는 짐 | 기내 반입 권장 |
| 배낭 D | 절대 분실하면 안 되는 서류·귀중품·전자기기 | 보호자가 항상 휴대 |

핵심 원칙:

- 가족 옷을 사람별 캐리어 하나에 몰지 않고 `28인치 A와 B에 절반씩` 나눈다.
- 28인치 한 개가 늦게 도착해도 세 사람 모두 갈아입을 옷이 있도록 구성한다.
- 보조배터리, 휴대폰, 태블릿·노트북과 귀중품은 위탁수하물에 넣지 않는다.
- 20인치가 기내 반입 기준을 충족하는지 출발 전에 항공사 허용 크기·무게를 최종 확인한다.

## 2. 여섯 구획 한눈에 보기

| 구획 | 위치 | 넣을 것 |
|---|---|---|
| 1 | 28인치 A 몸통 | 세 사람 4~5일분 옷, 첫 주 신발·수영용품 |
| 2 | 28인치 A 뚜껑 | 세면용품, 상비약 예비분, 도착 직후 생활용품 |
| 3 | 28인치 B 몸통 | 세 사람 나머지 옷, 여분 신발, 세탁용품 |
| 4 | 28인치 B 뚜껑 | 캠프 준비물, 문구, 공용 생활용품 |
| 5 | 20인치 C 몸통 | 가족 24시간용 옷과 잠옷, 얇은 겉옷 |
| 6 | 20인치 C 뚜껑 | 충전기, 어댑터, 기내용 편의용품, 서류 사본 |

## 3. 구획 1 — 28인치 A 몸통

`도착 후 가장 먼저 여는 캐리어`로 사용한다.

- [ ] 아버지 상의·하의·속옷·양말 4~5일분
- [ ] 윤찬 상의·하의·속옷·양말 4~5일분
- [ ] 소윤 상의·하의·속옷·양말 4~5일분
- [ ] 가족 잠옷 각 1벌
- [ ] 수영복·수영모 각 1세트
- [ ] 샌들 또는 슬리퍼 3켤레
- [ ] 모자 3개
- [ ] 접이식 우산 또는 우비

넣는 방법:

- 신발은 봉투에 넣어 `바퀴 쪽 아래`에 둔다.
- 옷은 사람별 파우치로 나누되 세 사람 파우치를 모두 이 구획에 넣는다.
- 가장 먼저 입을 잠옷과 속옷 파우치는 위쪽에 둔다.

## 4. 구획 2 — 28인치 A 뚜껑

`첫날 밤과 다음 날 아침에 바로 쓸 물품`을 넣는다.

- [ ] 칫솔·치약 3인분
- [ ] 샴푸·바디워시 등 첫 며칠분
- [ ] 선크림
- [ ] 벌레 기피제와 벌레 물린 뒤 바르는 약
- [ ] 체온계
- [ ] 해열제·소화제·감기약 등 상비약 예비분
- [ ] 밴드·소독용품
- [ ] 물티슈·휴지 여분
- [ ] 세탁망 1개
- [ ] 작은 장바구니 또는 에코백

주의:

- 액체류는 새지 않도록 각각 지퍼백에 넣고 뚜껑 안쪽에 고정한다.
- 비행 중 반드시 필요한 약은 여기에 넣지 말고 배낭에 넣는다.

## 5. 구획 3 — 28인치 B 몸통

`장기 체류용 나머지 옷`을 넣는다.

- [ ] 아버지 나머지 상의·하의·속옷·양말
- [ ] 윤찬 나머지 상의·하의·속옷·양말
- [ ] 소윤 나머지 상의·하의·속옷·양말
- [ ] 가족 잠옷 각 1벌
- [ ] 여분 운동화 또는 필요한 신발
- [ ] 여분 수영복
- [ ] 세탁물 보관 가방
- [ ] 여분 지퍼백

넣는 방법:

- 구획 1과 마찬가지로 세 사람 옷을 모두 넣어 수하물 지연 위험을 분산한다.
- 무거운 신발은 바퀴 쪽, 가벼운 옷은 손잡이 쪽에 둔다.
- 귀국 때 늘어날 짐을 고려해 완전히 압축하지 말고 약간의 빈 공간을 남긴다.

## 6. 구획 4 — 28인치 B 뚜껑

`캠프 시작 전 꺼내 정리할 물품`을 넣는다.

- [ ] 등원 가방 2개(납작하게 정리)
- [ ] 필통 2개
- [ ] 연필·지우개·필기구
- [ ] 노트
- [ ] 텀블러 2개
- [ ] 바람막이 2벌
- [ ] 학생증 목걸이 또는 케이스
- [ ] 이름표·네임펜
- [ ] 여분 마스크
- [ ] 멀티탭

주의:

- 태블릿·노트북은 이 구획에 넣지 않고 배낭에 휴대한다.
- 텀블러 안에는 양말이나 작은 문구를 넣어 공간을 활용할 수 있다.

## 7. 구획 5 — 20인치 C 몸통

`위탁수하물이 도착하지 않아도 세 사람이 하루를 보낼 수 있는 구성`으로 만든다.

- [ ] 아버지 갈아입을 옷·속옷·양말 1세트
- [ ] 윤찬 갈아입을 옷·속옷·양말 1세트
- [ ] 소윤 갈아입을 옷·속옷·양말 1세트
- [ ] 가족 잠옷 각 1벌
- [ ] 얇은 겉옷 또는 바람막이 각 1벌
- [ ] 최소 세면도구(기내 액체류 기준에 맞는 용량)
- [ ] 빈 텀블러 또는 물병 1개

넣는 방법:

- 사람별 1일 파우치 3개를 만들어 바로 꺼낼 수 있게 한다.
- 심야 RNF 체크인 후 이 캐리어 하나만 열어도 씻고 잘 수 있게 구성한다.

## 8. 구획 6 — 20인치 C 뚜껑

`기내와 도착 직후 필요한 비귀중품`을 넣는다.

- [ ] 휴대폰·태블릿 충전기와 케이블
- [ ] 멀티 충전기
- [ ] 영국식 Type G 변환 플러그
- [ ] 이어폰 또는 헤드폰
- [ ] 물티슈·휴지·마스크
- [ ] 아이 간식 일부
- [ ] 간단한 놀거리
- [ ] 볼펜
- [ ] 여권·보험·예약서 사본
- [ ] RNF 주소와 픽업 연락처 출력본

주의:

- 보조배터리와 현금은 이 구획이 아니라 배낭에 넣는다.
- 20인치가 탑승구에서 위탁 처리될 가능성에 대비해 고가 전자기기는 넣지 않는다.

## 9. 배낭 D — 보호자가 항상 휴대

배낭은 구획 수에 포함하지 않으며, 이동 중 보호자의 몸에서 떨어지지 않게 한다.

### 앞쪽 또는 바로 꺼낼 수 있는 칸

- [ ] 여권 3권
- [ ] 탑승권·서류 파우치
- [ ] 휴대폰 3대
- [ ] 지갑과 주 사용 카드
- [ ] 볼펜
- [ ] 물티슈·휴지
- [ ] 아이 간식

### 안쪽 지퍼 또는 보안 칸

- [ ] 예비 카드
- [ ] MYR·SGD 현금 일부
- [ ] 현지 유심 2개
- [ ] SGAC·MDAC 제출 확인본
- [ ] 여행자보험 증권
- [ ] JJTOUR 예약서와 RNF 주소

### 보호 패딩 칸

- [ ] 학생용 태블릿 또는 노트북 2대
- [ ] 보조배터리
- [ ] 기기용 충전 케이블 1세트

### 약 파우치

- [ ] 평소 복용약
- [ ] 아이용 해열제 등 비행·이동 중 필요한 최소 상비약
- [ ] 멀미약
- [ ] 알레르기약
- [ ] 처방전 또는 영문 약 정보

## 10. 짐 싸는 순서

1. 바닥에 `1~6` 구획 표지를 놓는다.
2. 모든 준비물을 먼저 해당 구획 옆에 모은다.
3. 28인치 A와 B에 세 사람 옷이 모두 들어갔는지 확인한다.
4. 배낭에 서류·현금·전자기기·보조배터리·필수 약을 먼저 넣는다.
5. 20인치 C만 열어도 24시간 생활 가능한지 확인한다.
6. 액체류 누수 방지와 이름 표시를 확인한 뒤 캐리어를 닫는다.
7. 캐리어별 무게를 재고, 두 28인치의 무게 차이가 크면 옷과 공용품을 재분배한다.
8. 각 캐리어 안쪽과 바깥쪽에 보호자 연락처를 표시한다.

## 11. 최종 확인표

- [ ] 28인치 A에 세 사람 옷이 모두 들어 있음
- [ ] 28인치 B에도 세 사람 옷이 모두 들어 있음
- [ ] 20인치 C에 세 사람의 24시간용 옷이 들어 있음
- [ ] 여권·현금·카드·전자기기·보조배터리는 배낭에 있음
- [ ] 위탁수하물에 보조배터리와 고가 전자기기가 없음
- [ ] 액체류는 이중 포장됨
- [ ] 여섯 구획을 사진으로 찍어 휴대폰에 저장함
- [ ] 캐리어 3개의 크기·무게가 항공사 허용 기준 안에 있음
- [ ] 가방마다 이름과 연락처가 표시됨
