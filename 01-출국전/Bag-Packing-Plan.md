# [출국 전] 가방별 배치 계획

Version: v2.4

작성일: 2026-07-22

준비물의 종류와 수량은 [Packing-Checklist.md](Packing-Checklist.md)에서만 관리한다. 이 문서는 `28인치 2개 + 20인치 1개 + 배낭 1개`의 운송 구분과 무게 배치만 정한다.

## 1. 가방 역할

| 가방 | 운송 | 역할 |
|---|---|---|
| 28인치 A | 위탁 | 아버지 의류를 기준 중량으로 삼고 나머지 물품 분산 |
| 28인치 B | 위탁 | 아이 의류·수건을 기준 중량으로 삼고 나머지 물품 분산 |
| 20인치 C | 기내 | 검색대에서 보통 꺼내지 않는 전자제품과 도착 직후 사용할 물품 |
| 배낭 D | 기내·보호자 휴대 | 검색대에서 꺼낼 대형 전자기기·서류·귀중품·배터리·필수 약 |

숙소 도착 후 모두 꺼내 정리하므로 캐리어 내부의 완벽한 분류보다 위탁 가능 여부와 무게 균형을 우선한다.

## 2. 여섯 구획

| 구획 | 위치 | 넣을 범주 |
|---|---|---|
| 1 | 28인치 A 몸통 | 아버지 의류를 먼저 넣고, 빈 공간에 가벼운 세탁망·우산과 무게 조정용 의류 |
| 2 | 28인치 A 뚜껑 | 가족 신발 일부, 연습용 배트, 멀티탭, 샤워헤드·마른 교체 필터 등 비교적 무거운 물품 일부 |
| 3 | 28인치 B 몸통 | 윤찬·소윤 의류와 수건을 먼저 넣고, 빈 공간에 잠옷·무게 조정용 의류 |
| 4 | 28인치 B 뚜껑 | 나머지 가족 신발, 첫 2~3일분 세면용품과 상비약 예비분, 윤찬 백팩·소윤 작은 나이키 백팩, 패드 파우치·문구·빈 텀블러 |
| 5 | 20인치 C 몸통 | 충전식 휴대용 선풍기 3대, 충전기·케이블, Type G 플러그 |
| 6 | 20인치 C 뚜껑 | 얇은 겉옷, 아이 여벌옷, 간식, 서류 사본 |

## 3. 배낭에만 넣을 것

- 여권과 탑승권
- 현금·주 카드·예비 카드
- 휴대폰
- 한 파우치에 모은 맥북과 태블릿 2대
- 전원을 완전히 끄고 하드 케이스에 넣은 닌텐도 스위치
- SGAC·MDAC 확인자료와 보험증권
- 새 20,000mAh 보조배터리
- 비행·이동 중 필요한 필수 약
- JJTOUR 예약서와 RNF 주소

보조배터리는 단자를 보호하고 개별 지퍼백에 넣는다.

맥북과 태블릿 파우치는 배낭의 등판 쪽에 세워 넣되 지퍼를 열면 바로 꺼낼 수 있게 한다. 보안검색대에서 분리하라는 안내가 있을 때 파우치 하나만 꺼내며, 그대로 두라는 안내가 있으면 꺼내지 않는다. 20인치 캐리어는 검색대에서 전자기기를 꺼내기 위해 눕혀 열지 않는다.

탑승 후 20인치 캐리어는 머리 위 선반에 넣고, 맥북·태블릿·닌텐도 스위치·보조배터리가 든 배낭은 앞좌석 아래에 넣는다. 이착륙 중에는 배낭을 발밑에 느슨하게 두지 않고 앞좌석 아래로 완전히 밀어 넣는다. 보조배터리는 머리 위 선반에 보관하지 않으며 기내에서 사용하거나 충전하지 않는다.

등원용 텀블러는 기내에서 사용하지 않고 빈 상태로 위탁한다. 출국일 물은 보안검색 후 구매한다.

## 4. 위탁수하물에 넣지 않을 것

- 여권·현금·카드
- 휴대폰·태블릿·노트북
- 보조배터리
- 필수 복용약
- 원본 예약·보험 서류

## 5. 무게 조정 순서

1. A 몸통에는 아버지 의류, B 몸통에는 아이 의류와 수건을 먼저 넣는다.
2. 신발·배트·멀티탭·샤워헤드·세면용품을 A와 B의 뚜껑에 나누어 1차 배치한다.
3. 28인치 A와 B의 무게를 각각 잰다.
4. 두 가방의 차이가 2kg을 넘으면 신발·세면용품·캠프용품을 먼저 옮기고, 마지막으로 의류로 미세 조정한다.
5. 항공사 위탁 허용 중량보다 최소 2kg 여유를 둔다.
6. 20인치의 기내 크기·중량 기준을 확인한다.
7. 최종 배치를 사진으로 남긴다.

## 6. 전원 사용

- 한국 멀티탭 1개와 Type G 변환 플러그 2개를 준비한다.
- Type G 하나는 공용 충전 거점, 하나는 침대 옆 단독 충전에 사용한다.
- 멀티탭은 휴대폰·태블릿·노트북 충전용으로만 사용한다.
- 드라이기·전기포트 같은 고출력 제품은 연결하지 않는다.

## 7. 최종 확인

- [ ] 준비물 수량은 Packing Checklist와 일치함
- [ ] 귀중품과 배터리는 위탁수하물에 없음
- [ ] 액체류는 이중 포장됨
- [ ] 두 28인치의 무게 차이가 2kg 이내임
- [ ] 각 가방이 항공사 허용 크기·중량 이내임
- [ ] 모든 가방에 이름과 연락처가 표시됨
