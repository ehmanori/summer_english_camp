# [출국 전] 가방별 배치 계획

Version: v2.0

작성일: 2026-07-22

준비물의 종류와 수량은 [Packing-Checklist.md](Packing-Checklist.md)에서만 관리한다. 이 문서는 `28인치 2개 + 20인치 1개 + 배낭 1개`의 운송 구분과 무게 배치만 정한다.

## 1. 가방 역할

| 가방 | 운송 | 역할 |
|---|---|---|
| 28인치 A | 위탁 | 의류와 무게 분산용 물품 |
| 28인치 B | 위탁 | 세면·건강·캠프용품 |
| 20인치 C | 기내 | 전자기기와 도착 직후 사용할 물품 |
| 배낭 D | 기내·보호자 휴대 | 서류·귀중품·배터리·필수 약 |

숙소 도착 후 모두 꺼내 정리하므로 캐리어 내부의 완벽한 분류보다 위탁 가능 여부와 무게 균형을 우선한다.

## 2. 여섯 구획

| 구획 | 위치 | 넣을 범주 |
|---|---|---|
| 1 | 28인치 A 몸통 | 아버지 의류, 가족 신발, 연습용 배트, 멀티탭 등 무거운 물품 |
| 2 | 28인치 A 뚜껑 | 윤찬·소윤 의류, 수건, 우산, 세탁망 |
| 3 | 28인치 B 몸통 | 물기를 빼 이중 포장한 샤워헤드·마른 교체 필터, 첫 2~3일분 세면용품과 상비약 예비분 |
| 4 | 28인치 B 뚜껑 | 윤찬 백팩·소윤 작은 나이키 백팩, 패드 파우치, 문구, 빈 텀블러 등 캠프용품 |
| 5 | 20인치 C 몸통 | 태블릿·노트북, 충전기, Type G 플러그 |
| 6 | 20인치 C 뚜껑 | 얇은 겉옷, 아이 여벌옷, 간식, 서류 사본 |

## 3. 배낭에만 넣을 것

- 여권과 탑승권
- 현금·주 카드·예비 카드
- 휴대폰
- SGAC·MDAC 확인자료와 보험증권
- 새 20,000mAh 보조배터리
- 비행·이동 중 필요한 필수 약
- JJTOUR 예약서와 RNF 주소

보조배터리는 단자를 보호하고 개별 지퍼백에 넣는다.

등원용 텀블러는 기내에서 사용하지 않고 빈 상태로 위탁한다. 출국일 물은 보안검색 후 구매한다.

## 4. 위탁수하물에 넣지 않을 것

- 여권·현금·카드
- 휴대폰·태블릿·노트북
- 보조배터리
- 필수 복용약
- 원본 예약·보험 서류

## 5. 무게 조정 순서

1. 준비물을 여섯 구획에 1차 배치한다.
2. 28인치 A와 B의 무게를 각각 잰다.
3. 두 가방의 차이가 2kg을 넘으면 신발·배트·멀티탭·옷을 옮긴다.
4. 항공사 위탁 허용 중량보다 최소 2kg 여유를 둔다.
5. 20인치의 기내 크기·중량 기준을 확인한다.
6. 최종 배치를 사진으로 남긴다.

## 6. 전원 사용

- 한국 멀티탭 1개와 Type G 변환 플러그 2개를 준비한다.
- Type G 하나는 공용 충전 거점, 하나는 침대 옆 단독 충전에 사용한다.
- 멀티탭은 휴대폰·태블릿·노트북 충전용으로만 사용한다.
- 드라이기·전기포트 같은 고출력 제품은 연결하지 않는다.

## 7. 최종 확인

- [ ] 준비물 수량은 Packing Checklist와 일치함
- [ ] 귀중품과 배터리는 위탁수하물에 없음
- [ ] 액체류는 이중 포장됨
- [ ] 두 28인치의 무게 차이가 2kg 이내임
- [ ] 각 가방이 항공사 허용 크기·중량 이내임
- [ ] 모든 가방에 이름과 연락처가 표시됨
