# B. Overview

## 1. 프로젝트 개요

### 1.1 프로젝트 성격

본 문서는 2026년 여름방학 기간 동안 가족 단위로 말레이시아 조호바루 `RNF`를 생활 거점으로 삼고, 캠프 종료 후 싱가포르를 마무리 여행 권역으로 활용하는 영어캠프형 장기체류 프로젝트의 실행 기준을 정리한 문서다.

핵심은 단순 관광이 아니라 다음 세 가지를 균형 있게 달성하는 것이다.

- 자녀의 영어 노출과 실사용 경험 확대
- 가족 단위 장기 해외 체류 운영 경험 축적
- 향후 장기 유학 또는 해외 교육 가능성 탐색

### 1.2 참가 가족

| 구분 | 내용 |
|---|---|
| 보호자 | 아버지 |
| 자녀1 | 김윤찬 (2015년생) |
| 자녀2 | 김소윤 (2017년생) |
| 참가 인원 | 총 3인 |

### 1.3 체류 기본 일정

| 구분 | 내용 |
|---|---|
| 출국일 | 2026-07-24 |
| RNF 체류 종료일 | 2026-08-22 |
| 귀국일 | 2026-08-25 |
| 체류 권역 | 말레이시아 조호바루 + 싱가포르 |
| 운영 방식 | 조호바루 RNF 29박 체류 + 싱가포르 3박 마무리 여행 |

### 1.4 운영 원칙

운영 우선순위는 `아이 컨디션 -> 생활 안정 -> 영어 노출 -> 체험 확장` 순으로 둔다.

## 2. 프로젝트 목적

### 2.1 자녀 교육 목적

- 영어에 대한 심리적 거리감 축소
- 생활 속 영어 사용 경험 확대
- 국제 환경에 대한 낯섦 완화
- 해외 체류 중 기본 생활 적응력 강화

### 2.2 가족 운영 목적

- 부모와 자녀가 함께하는 장기 해외 체류 실험
- 가족 단위 해외 생활 루틴 점검
- 향후 더 긴 해외 체류 가능성 사전 검토

## 3. 지역 선정 이유

### 3.1 조호바루를 거점으로 두는 이유

- 싱가포르 대비 숙박비와 생활비 부담이 낮다.
- 가족 단위 장기 체류에 적합한 숙소 선택 폭이 있다.
- Grab 중심 이동이 가능해 아이 동반 운영이 상대적으로 수월하다.
- 영어캠프 및 국제교육 자원 접근성이 있다.
- 싱가포르와의 연계가 가능해 체험 확장성이 높다.

### 3.2 싱가포르를 병행하는 이유

- 국제 도시 환경을 아이들이 직접 경험할 수 있다.
- 생활권 영어 노출 밀도를 높일 수 있다.
- 관광이 아니라 짧고 밀도 있는 체험일을 운영하기 좋다.
- 귀국 전 마무리 일정을 안정적으로 설계할 수 있다.

## 4. 현재 확정 전제

- 항공권 예약 완료
- 싱가포르 숙소 `Amara Singapore` 예약 완료
- 전체 체류 기간 확정
- 조호바루 `RNF` 체류 기간 확정: 2026-07-24 ~ 2026-08-22
- 조호바루 장기 숙소 확정: RNF 2 Bedroom
- 영어캠프는 거의 확정 상태이나 최종 시간표는 변동 가능

## 5. 프로젝트 성공 기준

이번 프로젝트의 성공은 다음 기준으로 판단한다.

- 아이들이 영어 사용 경험을 부담보다 재미로 기억할 것
- 장기 체류 중 생활 리듬이 무너지지 않을 것
- 부모의 피로 누적으로 일정 전체가 흔들리지 않을 것
- 무리한 관광보다 안정적인 루틴 운영이 우선될 것
- 귀국 후 향후 해외 교육 선택지를 현실적으로 평가할 수 있을 것
